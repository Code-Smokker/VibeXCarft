# VibeXCraft

VibeXCraft is a full-stack AI Mesh platform designed to enable real-time collaborative coding with multi-agent AI integration. It fosters a shared workspace environment where developers can engage in live pair-programming sessions enhanced by video calls and intelligent AI assistance.

## Features

- **Real-Time Collaborative Coding**: Code together in real-time with seamless synchronization.
- **Multi-Agent AI Mesh**: Integrate multiple AI agents to assist with coding, debugging, and suggestions.
- **Shared Workspaces**: centralized environments for teams to collaborate effectively.
- **Live Pair-Programming**: Integrated video calls to facilitate face-to-face communication during coding sessions.
